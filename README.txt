САЙТ-ПРИГЛАШЕНИЕ «МАТВЕЙ & ДИЁРА»

Файлы:
- friends.html — версия для друзей
- family.html — версия для семьи
- index.html — меню предпросмотра
- assets/ — фотографии
- styles.css — оформление
- script.js — плавное появление блоков

Как посмотреть:
1. Распакуйте архив.
2. Откройте index.html в браузере.
3. Выберите нужную версию.

Как опубликовать бесплатно:
- Netlify Drop: перетащите распакованную папку на страницу Netlify Drop.
- GitHub Pages: загрузите всю папку в репозиторий.
- Любой хостинг: загрузите все файлы без изменения структуры.
